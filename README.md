# BBKing

想分享一些比较值得玩味的话术和观点，原本想叫『Slogan』，但是感觉不如BBKing来的霸气，『 You can you up，No can no BB 』😁

BBKing这个词来源于《奇葩说》，以这个名字命名也并不是让自己成为杠精，而是想时刻告诫自己不管对于什么事情要多思考，要尽可能仔细、客观地理解和描述事情本身。
